Komast fram hjá password í PostgreSQL

1. Opna pg_hba.config file
	- (staðsetningin hjá mér: D:\Program Files (Megas)\PostgreSQL\9.5\data)
	- Gætir þurft að keyra notpade með 'run as admin'

2. Þar sem það er uncomment-að, og undir dálknum METHOD, breyta úr 'md5' yfir í 'trust' svo save-a

3. Opna cmd í staðsetningunni þar sem þú ert með forritið psql
	- (hjá mér: D:\Program Files (Megas)\PostgreSQL\9.5\bin)

3. Skrifar: psql
	+ Ef þú færð eitthvað villu þá gæturu þurft að runna sem annar notandi
	3.2 Skrifa psql -U <notenda_nafn>
		- hjá mér: psql -U postgres

4. Breyta passwordi -> ALTER USER "user_name" WITH PASSWORD <your_new_password>

5. Breyta aftur í sömu stöðu frá skrefi 2.

6. Gætir þurft að endurræsa tölvuna.
